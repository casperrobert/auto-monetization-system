EINFACH-START (macOS)
1) Docker Desktop starten (einmal).
2) ZIP entpacken und Ordner öffnen.
3) Rechtsklick auf 'Start_macOS.command' -> Öffnen -> nochmal 'Öffnen' bestätigen.
   (Nur beim ersten Mal. Danach reicht Doppelklick.)
4) Browser: http://localhost:3000
Stoppen: Doppelklick auf 'Stop_macOS.command'.
